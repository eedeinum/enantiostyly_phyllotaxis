These scripts are to analyze data about the phyllotactic orientation and flower handedness in Cyanella alba. These scripts and data belong to the publication "Spiral phyllotaxis predicts left-right asymmetric growth and style deflection in mirror-image flowers of Cyanella alba", Roberts et al 2025, Nature Communications. Please cite the manuscript when using these scripts or data for any purpose.

This repository has been filed at Zenodo with the following DOI: 10.5281/zenodo.14989473

All scripts are created by Eva E. Deinum
